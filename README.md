# About Me Static Site

This repository contains a single-page personal "About Me" website built to satisfy the requirements of Assignment 3 for hosting a static webpage on AWS S3. The page highlights a University of Ottawa Computer Science student, their education, projects, skills, and contact information.

## Preview

Open `index.html` in a browser to view the site locally.

## Deployment

To deploy the site to an AWS S3 bucket for static web hosting:

1. Create (or reuse) an S3 bucket with a unique name and enable **Static website hosting**.
2. Upload `index.html` and `styles.css` to the bucket.
3. Set the bucket policy and object permissions to allow public read access if required by your instructor.
4. Use the S3 website endpoint URL provided in the console to view the live site.

Remember to include the link to your hosted page when submitting the assignment report.
